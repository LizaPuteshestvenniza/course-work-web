# course-work-web

В рамках текущего апдейта нами реализованы следующие фичи:
1. Header сайта
2. Home page
3. Каталог товаров
4. Корзина
5. Выбор товаров в корзину / удаление из корзины
6. Авторизация
7. Sign in 
8. Username
